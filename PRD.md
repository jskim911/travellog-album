# Product Requirements Document (PRD)
# 여행 앨범 앱 개선 프로젝트

## 📋 문서 정보
- **프로젝트명**: TravelLog Album Enhancement
- **버전**: 2.0
- **작성일**: 2025-12-14
- **상태**: Draft

---

## 🎯 프로젝트 개요

### 목적
기존 여행 앨범 앱을 개선하여 사진 관리뿐만 아니라 여행 일정 관리, 비용 추적, 스토리보드 생성 등 종합적인 여행 관리 플랫폼으로 전환

### 핵심 가치 제안
- AI 기반 자동화로 여행 기록의 편의성 극대화
- 사진, 일정, 비용을 통합 관리하는 올인원 플랫폼
- 지인 간 공유를 위한 프라이빗 커뮤니티

---

## 👥 사용자 페르소나

### 주요 타겟
- **연령대**: 20-40대
- **특징**: 여행을 즐기고 기록을 남기는 것을 좋아하는 사용자
- **니즈**: 
  - 여행 사진을 체계적으로 정리하고 싶음
  - 여행 비용을 추적하고 관리하고 싶음
  - 여행 추억을 멋진 형태로 보관하고 공유하고 싶음

---

## 🎨 주요 기능 요구사항

### 1. 인증 및 사용자 관리

#### 1.1 회원가입 시스템
- **승인 기반 회원가입**
  - 회원가입 신청 시 이름, 이메일 입력
  - 관리자 승인 후 계정 활성화
  - 지인만 사용하는 프라이빗 커뮤니티 구현

#### 1.2 로그인 시스템
- 이메일/비밀번호 기반 로그인
- 로그인 성공 시 환영 배너 표시: "{이름}님 여행앱에 오신 것을 환영합니다!"

#### 1.3 데이터 관리 정책
- **자동 삭제**: 회원별 데이터 30일 보관 후 자동 삭제
- **계정 삭제**: 회원 아이디 삭제 시 모든 연관 데이터 자동 삭제
- **이미지 최적화**: 저장 시 이미지 용량 자동 최적화

---

### 2. 사진 업로드 및 관리

#### 2.1 사진 업로드
- 다중 사진 업로드 지원
- 업로드 시 AI 기반 자동 분석
  - 장소 인식
  - 여행 소감/느낌 문구 10개 추천 (Gemini 2.5 Flash)

#### 2.2 메타데이터 입력
- 여행 장소 입력/수정
- AI 추천 문구 중 선택 또는 직접 입력
- 날짜 자동 인식 및 수동 조정

---

### 3. 갤러리 기능

#### 3.1 사진 분류 및 표시
- **카테고리 뷰**:
  - 날짜별 (하루 단위)
  - 장소별
- 그리드 레이아웃으로 사진 표시

#### 3.2 사진 관리 기능
- **단일 선택**: 개별 사진 삭제/다운로드
- **멀티 선택**: 여러 사진 동시 삭제/다운로드
- 사진 상세 보기 (메타데이터 포함)

---

### 4. 여행 스토리보드

#### 4.1 자동 생성
- 하루 단위별 사진 기반 스토리보드 자동 생성
- 사진, 장소, 소감을 조합한 시각적 레이아웃
- AI가 스토리 흐름 최적화

#### 4.2 편집 및 내보내기
- 생성된 스토리보드 편집 가능
- PDF 파일로 내보내기
- 고품질 인쇄용 포맷 지원

---

### 5. 여행 일정 로드맵

#### 5.1 일정 입력
- **입력 항목**:
  - 여행 구간
  - 루트 (출발지 → 목적지)
  - 방문 장소
  - 식당
  - 비용
  - 비고

#### 5.2 로드맵 시각화
- 그래픽/이미지 형태로 루트 표시
- 방문 장소, 식당 위치 표시
- 인터랙티브 맵 또는 타임라인 뷰

#### 5.3 비용 관리
**영수증 인식**:
- 영수증 이미지 업로드
- AI 기반 OCR로 내역, 금액 자동 추출
- 영수증 이미지 별도 저장

**수동 입력**:
- 영수증 없는 경우 직접 입력
- 항목, 금액, 날짜, 카테고리 입력

**비용 추적**:
- 실시간 누적 비용 계산 및 표시
- 카테고리별 지출 분석
- PDF 리포트 내보내기 (항목별 내역, 금액, 영수증 포함)

---

### 6. 여행 이모지 생성

#### 6.1 이모지 선택 및 편집
- 갤러리에서 사진 선택
- 원형 크롭 도구 제공
- 크기 및 위치 조정

#### 6.2 AI 이모지 생성
- 선택한 사진 기반 8개 다양한 이모지 생성
- 8개 이모지를 하나로 합친 컬렉션 이미지 생성
- 재생성 기능 (마음에 들지 않을 경우)

#### 6.3 다운로드
- 개별 이모지 다운로드
- 전체 컬렉션 다운로드
- PNG 투명 배경 지원

---

## 🤖 AI 통합 요구사항

### Gemini 2.5 Flash 활용
1. **사진 분석**: 장소, 분위기 인식
2. **문구 추천**: 여행 소감 10개 생성
3. **영수증 OCR**: 텍스트, 금액 추출
4. **스토리보드 생성**: 사진 배치 및 스토리 구성 최적화
5. **이모지 생성**: 사진 기반 다양한 스타일의 이모지 생성

---

## 💾 데이터 관리 요구사항

### 데이터베이스 스키마
- **Users**: 사용자 정보, 승인 상태
- **Photos**: 사진 메타데이터, URL, 날짜, 장소, 소감
- **Itineraries**: 여행 일정 정보
- **Expenses**: 비용 내역, 영수증 이미지
- **Storyboards**: 생성된 스토리보드 데이터

### 저장소
- **Firebase Storage**: 사진, 영수증, PDF 파일
- **이미지 최적화**: 
  - 원본: 최대 2MB
  - 썸네일: 200KB 이하
  - WebP 포맷 사용

---

## 🎨 UI/UX 요구사항

### 디자인 원칙
- **모던하고 직관적**: 깔끔한 인터페이스
- **반응형**: 모바일, 태블릿, 데스크톱 지원
- **빠른 로딩**: 최적화된 이미지 로딩
- **접근성**: WCAG 2.1 AA 준수

### 주요 화면
1. 로그인/회원가입
2. 대시보드 (환영 배너, 최근 여행)
3. 사진 업로드
4. 갤러리 (날짜별/장소별)
5. 스토리보드 생성/편집
6. 여행 로드맵
7. 비용 관리
8. 이모지 생성

---

## 🔒 보안 요구사항

- Firebase Authentication 사용
- 관리자 승인 시스템
- 사용자별 데이터 격리
- 이미지 업로드 크기 제한 (10MB)
- XSS, CSRF 방어

---

## 📊 성공 지표

- 회원가입 신청 → 승인 시간: 24시간 이내
- 사진 업로드 성공률: 99% 이상
- AI 분석 응답 시간: 5초 이내
- 페이지 로드 시간: 3초 이내
- 사용자 만족도: 4.5/5 이상
