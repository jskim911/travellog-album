제목 : 여행앱 개선 아이디어 수립

1. 전체 레이아웃 변경 및 로그인 기능 개선
  - 무비메이커 생성 기능 삭제
  - AI모델은 gemini 2.5flash 모델 적용할 것
  - 가입자 아이디어별로 화면 생성하고 데이터 저장 관리
  - 로그인되면 "ㅇㅇ님 ㅇㅇ 여행앱에 들온것을 환영합니다!" 배너창에 만듬
  - 회원별 데이터 관리는 30일간 저장하고 이후 전체 자동 삭제 됨
  - 회원아이디가 삭제되면 자동으로 모든 데이터 삭제
  - 데이터베이스에 저장시 이미지 데이터 용량은 최적화하여 너무 크지 않게 변환하여 저장
  - 로그인 창에서 회원가입을 누르면 관리자에게 승인 요청 전달(지인에만 사이트 사용목적)
  - 회원가입 모달창에는 회원 이름, 이메일을 작성하고 요청버튼을 누른다.
  - 관리자가 승인하면 회원가입 진행절차 수행
2. 사진입력 기능 개선
  - 사진 선택하고 로드할 때 AI가 여행 장소와 느낌을 입력할 때 여행에서 가질수
    있는 소감, 느낌 문구를 10개 추천
3. 갤럭리 창에 개선
  - 하루 단위, 장소별 카테고리로 보이도록 한다.
  - 등록된 사진은 삭제, 다운로드 할 수 있도록 한다.
  - 멀티 선택하여 삭제, 다운로드가 되어야 한다.
4. 여행 스토리보드 만들기
  - 하루 단위별로 저장된 사진을 기반으로 작성된 소감, 장소 특성을 반영하여 자동 생성하여 저장
  - 내보내기 버튼을 누르면 PDF파일로 작성하여 내보내기 한다.
5. 여행일정 로드맵 생성
  - 로드맵 작성을 위한 입력창을 생성한다.
  - 입력양식은  여행 구간, 가는 루트(출발지, 목적지), 방문장소, 식당, 비용 등록, 비고
  - 로드맵 생성 버튼을 누르면 간단한 그래픽으로 그려서 별도의 고장창을 만들어 
    루트, 방문장소, 식당 그랙 또는 이미지 형태로 보여준다.
  - 양식은 테이블 양식, 너가 더 좋은 방법 있으면 그것으로 할 것
  - 비용 등록 방법은 이미지 영수증을 넣으면 내역, 금액을 자동으로 인식하여 저장, 영수증 이미지 별도 저장
  - 영수증 없을 경우 비용등록은 직접 작성할 수 있도록 추가한다.
  - 비용등록 저장하면 자동으로 현재까지 비용을 계산하여 보이도록 한다. 
  - 비용 출력버튼을 누르면 지금까지 지출한 항목별 지출내역, 금액, 영수증 등을 PDF파일로 내보내기 한다.
6. 여행 이모지 생성 기능
  - 갤러리에서 이모지를 생성할 사진을 선택한다.
  - 선택된 이모지를 동그라미 모양의 잘라내기 편집이 되도록 한다
  - 이모지 생성버튼을 누르면 사진 바탕의 다양한 이모지를 8개 생성하고, 8개 이모지를 하나로 담은 이모지도 생성한다.
  - 재생성 및  다운로드 기능을 할 수 있도록 한다.
