# � 여행앱 개선 아이디어

## 1. 전체 레이아웃 변경 및 로그인 기능 개선

* 무비메이커 생성 기능을 삭제합니다.
* AI 모델은 **Gemini 2.5 Flash** 모델을 적용합니다.
* 가입자 아이디어별로 화면을 생성하고 데이터를 저장 관리합니다.
* 로그인되면 배너창에 **"ㅇㅇ님 ㅇㅇ 여행앱에 들온것을 환영합니다!"** 문구가 나타납니다.
* 회원별 데이터는 **30일간 저장**되고, 이후 전체가 **자동으로 삭제**됩니다.
* 회원 아이디가 삭제되면 자동으로 모든 데이터가 삭제됩니다.
* 데이터베이스에 저장 시 이미지 데이터 용량은 최적화하여 너무 크지 않게 변환하여 저장합니다.
* 로그인 창에서 회원가입을 누르면 관리자에게 승인 요청이 전달됩니다 (지인에게만 사이트 사용 목적).
* 회원가입 모달창에는 **회원 이름, 이메일**을 작성하고 요청 버튼을 누릅니다.
* 관리자가 승인하면 회원가입 진행 절차를 수행합니다.

## 2. 사진 입력 기능 개선

* 사진을 선택하고 로드할 때, AI가 여행 장소와 느낌을 입력할 때 여행에서 가질 수 있는 **소감, 느낌 문구를 10개 추천**합니다.

## 3. 갤러리 창 개선

* 사진이 하루 단위, 장소별 카테고리로 보이도록 합니다.
* 등록된 사진은 **삭제, 다운로드** 할 수 있도록 합니다.
* **멀티 선택**하여 삭제, 다운로드가 가능해야 합니다.

## 4. 여행 스토리보드 만들기

* 하루 단위별로 저장된 사진을 기반으로 작성된 소감, 장소 특성을 반영하여 **자동으로 생성**하고 저장합니다.
* **내보내기 버튼**을 누르면 **PDF 파일**로 작성하여 내보내기 합니다.

## 5. 여행 일정 로드맵 생성

* 로드맵 작성을 위한 입력창을 생성합니다.
* 입력 양식은 **여행 구간, 가는 루트(출발지, 목적지), 방문 장소, 식당, 비용 등록, 비고**입니다.
* **로드맵 생성 버튼**을 누르면 간단한 그래픽으로 그려서 별도의 고정창을 만들고 루트, 방문 장소, 식당을 그래픽 또는 이미지 형태로 보여줍니다.
    * 양식은 테이블 양식으로 하거나, 더 좋은 방법이 있으면 그것으로 할 수 있습니다.

### 비용 등록 방법

* **이미지 영수증**을 넣으면 내역, 금액을 자동으로 인식하여 저장하고, 영수증 이미지는 별도로 저장합니다.
* 영수증이 없을 경우 비용 등록은 직접 작성할 수 있도록 추가합니다.
* 비용 등록 저장 시 자동으로 현재까지의 비용을 계산하여 보이도록 합니다.
* **비용 출력 버튼**을 누르면 지금까지 지출한 항목별 지출 내역, 금액, 영수증 등을 **PDF 파일**로 내보내기 합니다.

## 6. 여행 이모지 생성 기능

* 갤러리에서 이모지를 생성할 사진을 선택합니다.
* 선택된 이모지를 **동그라미 모양으로 잘라내기 편집**이 되도록 합니다.
* **이모지 생성 버튼**을 누르면 사진 바탕의 **다양한 이모지를 8개 생성**하고, 이 8개 이모지를 하나로 담은 이모지도 생성합니다.
* **재생성** 및 **다운로드** 기능을 할 수 있도록 합니다.
